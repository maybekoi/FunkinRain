# Rain Engine [ALPHA]

A FNF Engine built from the ground up!

# Features

* Rain Input - An all new input system!

* Swag Options menu!!

* DownScroll

* MiddleScroll

* Botplay

* Ghost Tapping (off by default)

* Opponent Note Visibility Toggle

* Polymod support!

* Character and Stage .hscript files!

* and more!
